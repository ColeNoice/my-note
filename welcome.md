      ## 如何发布文章
      
      1. 在 Obsidian 中创建 markdown 文件
      2. 在文件开头添加 front matter，并设置 `share: true`
      3. 提交并推送到 GitHub
      4. GitHub Actions 会自动同步到博客
      
      开始写作吧！
